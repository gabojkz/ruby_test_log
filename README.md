# Parse

Takes as an argument a log file and outputs as a result a total count of pages visited.

## Requirements

1. Rspec for testing


## Usage

```bash
ruby parser.rb <log_file_name> [order=desc], [unique=false]
```


## Example

```bash
ruby parser.rb webserver.log desc false
```
